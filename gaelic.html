Prepositional pronouns refer to the combination of a preposition + pronoun (such as 'on me,' 'at you,' 'in it') etc. There are several of these combinations in Gaelic, so I created this quiz in python to test on various combinations. I thought it might be of use to other learners, so I transferred it to javascript. Many thanks to taic.me.uk
